# Project Manager — Fixed Clean Version

Use this as a NEW IntelliJ project. Do not merge it into the incomplete old folder.

## Exact package

`com.axsosacademy.projectmanager`

## Exact main class

`ProjectmanagerApplication.java`

## Before running

1. Extract the ZIP.
2. In IntelliJ choose File -> Open.
3. Open the extracted `ProjectManager_FIXED` folder.
4. Open `src/main/resources/application.properties`.
5. Replace `YOUR_MYSQL_PASSWORD` with your actual MySQL password.
6. Reload Maven.
7. Run `ProjectmanagerApplication.java`.
