# Alfred Bot

## Assignment

This assignment builds a simple Java class called `AlfredQuotes`.

The class contains methods that return different Alfred-style responses, such as greetings, date announcements, and responses based on a conversation.

## Concepts Practiced

- Creating multiple Java files
- Using a test file with the `main` method
- Creating an object from another class
- Calling methods using dot notation
- Returning `String` values from methods
- Using `import java.util.Date`
- Using `String` methods like `indexOf()`
- Using `if`, `else if`, and `else`
