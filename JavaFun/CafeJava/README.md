\# Cafe Java



\## Assignment



This assignment simulates the basic logic for a cafe ordering app using Java.



The program stores menu prices, customer names, and order status values. It then prints messages depending on whether each customer's order is ready or still pending.



\## Concepts Practiced



\- Java variables

\- String

\- double

\- boolean

\- String concatenation

\- if / else statements

\- Printing output with System.out.println()



\## Menu Items



The program includes prices for:



\- Mocha

\- Drip Coffee

\- Latte

\- Cappuccino



\## Customers



The program includes the following customers:



\- Shatha

\- Ahmad

\- Sali

\- Adam



\## Program Features



The program does the following:



1\. Prints a welcome message.

2\. Checks if Sali's coffee order is ready.

3\. Checks if Ahmad's cappuccino order is ready.

4\. Displays Ahmad's total if his order is ready.

5\. Calculates the total for Sali ordering two lattes.

6\. Updates Sali's order status to test the if statement.

7\. Calculates the difference Adam owes after being charged for coffee instead of a latte.



\## Example Output

Sali, your order will be ready shortly

Your total is $2.5  

Your total is $13.0
