# CafeUtil

## Assignment

This assignment builds a small Java utility class for a cafe application.

The project uses two Java files:

- `CafeUtil.java`
- `TestCafe.java`

`CafeUtil.java` contains the methods and logic.

`TestCafe.java` contains the `main` method and is used to test the methods.

## Concepts Practiced

- Creating methods
- Returning values from methods
- Using arrays
- Using `ArrayList`
- Using loops
- Using user input
- Creating objects from another class
- Calling methods using dot notation
- Testing code with a separate test file

## Methods Implemented

### `getStreakGoal()`

Calculates the total number of drinks needed after 10 weeks.

It adds the numbers from 1 to 10.

Expected result:

55
