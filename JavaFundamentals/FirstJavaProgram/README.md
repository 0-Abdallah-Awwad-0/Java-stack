# First Java Program

## Assignment Description

This assignment is about creating a simple Java program that prints my name, age, and hometown.

## What the Program Prints

- My name
- My age
- My hometown

## File

FirstJavaProgram.java

## Output

My name is Abdallah Awwad
I am 24 years old
My hometown is Ramallah, Palestine

## How to Compile

```bash
javac FirstJavaProgram.java
```

## How to Run

```bash
java FirstJavaProgram
```

## Notes

The program uses the main method and System.out.println() to print each line.
